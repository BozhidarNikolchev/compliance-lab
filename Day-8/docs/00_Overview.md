# Overview
**NIST Cybersecurity Framework (CSF) 2.0** is an outcome-based framework with six functions: **Govern (GV), Identify (ID), Protect (PR), Detect (DE), Respond (RS), Recover (RC)**. It provides a taxonomy of outcomes, **Organizational Profiles (Current/Target)**, and **CSF Tiers** for describing maturity.
