# Security Audit Checklist

## Access Control
- [ ] MFA enforced  
- [ ] Least privilege applied  
- [ ] Access review performed quarterly  

## Logging & Monitoring
- [ ] SIEM active  
- [ ] Logs retained 90–365 days  

## Incident Response
- [ ] IR plan updated  
- [ ] Roles defined  
- [ ] Tabletop exercise in last 12 months  

## Asset Management
- [ ] Asset inventory complete  
- [ ] Owners assigned  

## Risk Management
- [ ] Risk register maintained  
- [ ] Treatment plans approved  

## Vendor Management
- [ ] Vendor list documented  
- [ ] Risk levels assigned  
- [ ] DPAs signed
