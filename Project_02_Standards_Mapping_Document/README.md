# Standards Mapping Document

This directory contains a complete, unified standards mapping package covering:

- ISO 27001:2022  
- NIST Cybersecurity Framework 2.0  
- SOC 2 Trust Services Criteria  
- PCI-DSS v4.0  
- GDPR (Articles 5, 25, 30, 32, 33)

The purpose of this folder is to centralize mapping between external frameworks and internal controls to reduce duplicate audit work and support evidence reuse.

## Files included

- `standards_mapping_overview.md` — introduction, scope, frameworks  
- `mapping_process_steps.md` — instructions for how mapping was performed  
- `control_mapping_matrix.csv` — unified control mapping table  
- `evidence_alignment_table.md` — what evidence satisfies which controls  
- `assets/` — diagrams and future screenshots

## Goal

To create a clean, portfolio‑ready mapping package demonstrating:
✅ regulatory understanding  
✅ cross‑framework alignment  
✅ audit preparation capability  
✅ documentation skills
