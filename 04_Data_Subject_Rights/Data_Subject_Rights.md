
# Topic 4 – Data Subject Rights (DSRs)


## Overview
This document summarizes the GDPR Data Subject Rights and the standard DSAR workflow used in compliance operations.

## 1. The Eight GDPR Data Subject Rights
1. Right to Access
2. Right to Rectification
3. Right to Erasure (Right to Be Forgotten)
4. Right to Restrict Processing
5. Right to Data Portability
6. Right to Object
7. Rights related to Automated Decision-Making
8. Right to Be Informed

## 2. DSAR Workflow
1. Request received
2. Identity verification
3. Data discovery and collection
4. Review and classification
5. Prepare response
6. Respond within 30 days
7. Store evidence

## 3. Evidence Examples
- Identity verification screenshot
- Data export CSV
- Communication logs
- Deletion confirmation
- Final DSAR response file

## 4. Tools Commonly Used
- OneTrust / TrustArc
- Jira / ServiceNow
- Azure AD / Okta
- Internal CRM systems
