# Evidence Checklist – Topic 7 (ISO 27001 Controls)
**Author:** Bozhidar Nikolchev  

Use this list when preparing for audits or internal reviews.

## Access & Authentication
- [ ] MFA enforcement report (tenant‑wide + break‑glass accounts)
- [ ] Password policy configuration export
- [ ] Privileged role assignments review and approvals

## Endpoint & Configuration
- [ ] MDM compliance report (encryption, firewall, OS version)
- [ ] CIS benchmark scan results or screenshots
- [ ] EDR/AV agent coverage report

## Backup & Recovery
- [ ] Backup job success reports (last 30 days)
- [ ] Periodic restore test records (dates, RTO/RPO results)
- [ ] Immutable/offline copy evidence

## Logging & Monitoring
- [ ] SIEM alert catalog and recent detections export
- [ ] Log retention configuration (hot/cold tiers)
- [ ] Use case mapping to risks/controls

## Training & Awareness
- [ ] Security training completion report
- [ ] Phishing simulation results
- [ ] Policy acknowledgement records
