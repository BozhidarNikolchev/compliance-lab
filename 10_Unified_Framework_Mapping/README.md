# Topic 10 – Unified Framework Mapping

This folder contains mapping material for major security & compliance frameworks:
- ISO 27001:2022  
- NIST Cybersecurity Framework 2.0  
- PCI-DSS v4.0  
- GDPR  
- SOC 2 Trust Services Criteria  

## 📂 Files Included
- `unified_framework_mapping.md` — main document  
- `framework_mapping_table.csv` — control mapping table  
- `assets/` — storage for diagrams, screenshots, evidence  

## ✅ Purpose  
To create unified internal controls and reduce duplicate audit work across frameworks.
